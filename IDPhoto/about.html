关于证照之星

Bundle ID：com.IDPhoto.ios ｜ 平台：iOS（SwiftUI）

核心价值主张

我们提供完全免费、高清无水印、操作简单、效果专业的证件照制作服务，覆盖学生、求职者、备考人员及各类签证办理人群。

功能亮点

  • 端侧AI抠图：基于 Apple Vision（iOS 15+），图片不离开设备，隐私更安全。
  • 规格全面：内置数百种考试与签证尺寸，精准头部比例与背景要求。
  • 拍摄引导与质检：人脸对齐、姿势校正、光线与清晰度检测，减少重拍。
  • 背景与美颜：红/白/蓝等背景；磨皮提亮（免费）；高级祛痘/塑形（Pro）。
  • 正装模板与云相册（Pro）：一键换正装；加密历史记录便捷复用。

技术方案

  • 架构与语言：iOS 原生 Swift + SwiftUI，图片处理采用 Core Image / Core Graphics。
  • AI：Vision 人像分割，必要时可引入轻量 Core ML 备选（如 MODNet / U^2-Net）。
  • 依赖：Swift Package Manager（SPM）。
  • 商业化支撑：StoreKit 2 内购、Google AdMob 广告、（可选）友盟统计。

隐私与合规

我们坚持“端上处理”，您的照片与人像数据不会上传或存储在服务器。若启用数据分析，仅收集匿名使用数据用于改进体验。详见《隐私政策》与《用户服务条款》。
