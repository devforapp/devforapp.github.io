测速仪 Speedometer

专业设备测试与速度测量

基于 GPS 与传感器的实时仪表盘、数据记录与导出，隐私友好端侧处理

核心功能

GPS 速度

使用 Core Location 获取实时速度、平均/最高速度，支持 km/h、mph、knots。

传感器测试

Core Motion 加速度计、陀螺仪、磁力计与设备姿态实时显示，支持校准。

图表与仪表盘

SwiftUI 自定义仪表盘与 Charts 实时波形图，数据到 UI 流畅绑定与动画。

数据记录与导出

SwiftData/Core Data 保存历史报告，支持 CSV/PDF 导出与文件 App 共享。

订阅与试用

StoreKit 2 订阅（周/月/年），提供 3 天免费试用；专业版移除广告并解锁全部功能。

隐私与权限

端侧本地处理；仅在用户授权后访问定位与运动权限；遵循 App Tracking Transparency。

技术规格

系统要求

  • iOS 14.0 或更高版本
  • 定位服务用于速度测量
  • 运动与方向传感器用于测试
  • 蓝牙用于可选设备测试

核心技术

  • Swift + SwiftUI + MVVM
  • Core Location / Core Motion / AVFoundation / Core Bluetooth
  • StoreKit 2 / Google AdMob
  • SwiftData/Core Data
